What is Cyber Security

At the beginning, we will have:

  • An introduction about security and its importance
  • How the Internet began
  • How to start in security
  • Learn basics of networks
  • Talk about ARP

This is in order to understand the OSI model and what each layer does and attacks that can happen.

After that, we will discuss some concepts such as the differences between the types of malwares and also we will start to deal with linux and how to use the terminal. Then penetration testing methodology,and every phase in it (information gathering - scanning - exploitation-privilege escalation).
And we will be ready to apply all of this and solve machines.
